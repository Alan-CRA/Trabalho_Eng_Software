# User Stories

## US 1 - Pesquisa de filmes

### Descrição
Como usuário do sistema, quero pesquisar os filmes disponiveis em streaming, para que eu possa assisti-los.

###	Critérios de aceitação:
1. O usuário deve ser capaz de filtrar a pesquisa por categoria, duração, disponibilidade no streaming, etc.
2. A sinopse deve ser visível antes que o usuário acesse a página de detalhamento do filme.

## Prioridade
Alta

## Estimativa
5 pontos

## Us2 - Detalhamento do filme

### Descrição
Como usuário do sistema, quero acessar uma página que contenha todas as informações do filmes, para que eu possa decidir se irei assistir o filme.

### Critérios de aceitação:
1. A página deve conter título, ano, duração, gênero e elenco.
2. Deve conter a sinopse completa.
3. Deve conter links para plataformas de streaming disponíveis.

## Prioridade
Alta

## Estimativa
3 pontos
 
## Us3 - Autenticação de usuários
### Descrição
Como futuro usuário, quero conseguir criar uma conta e realizar login no sistema.

### Critérios de aceitação:
1. O usuário deve poder se cadastrar com e-mail e senha.
2. O sistema deve validar credenciais antes de permitir o login.
3. O usuário deve poder recuperar a senha em caso de esquecimento.

## Prioridade
Alta

## Estimativa
3 pontos

## US4 - Avaliação de Filmes

### Descrição
Como usuário autenticado, quero avaliar os filmes que assisti com uma nota e comentário, para que eu possa registrar minha opinião e ajudar outros usuários a decidir o que assistir.

### Critérios de aceitação:
1. O usuário deve estar autenticado para poder avaliar um filme.
2. O usuário deve poder atribuir uma nota de 1 a 10 estrelas ao filme.
3. O usuário deve poder adicionar um comentário opcional à sua avaliação.
4. O usuário deve poder editar ou excluir sua avaliação a qualquer momento.
5. A avaliação do usuário deve ser exibida na página de detalhes do filme.
6. O usuário deve poder visualizar todas as suas avaliações em seu perfil.
7. A média das avaliações dos usuários deve ser exibida na página do filme.
8. Usuários não autenticados devem ver um convite para fazer login antes de avaliar.

## Prioridade
Alta

## Estimativa
5 pontos
